# تطبيق عيادة بيطرية - متجر + دفتر لقاحات

تطبيق React Native (Expo) باللغة العربية بالكامل، مع:

- متجر منتجات تجريبية
- تصنيفات + فلترة حسب العلامة التجارية
- سلة حجز
- إرسال طلب الحجز عبر البريد إلى: `salamadil1233@gmail.com`
- دفتر رقمي للحيوانات واللقاحات
- جدولة تذكير تلقائي لموعد اللقاح عبر إشعار محلي
- الدفع عند الاستلام فقط

## التشغيل

```bash
npm install
npm run start
```

## ملاحظة مهمة

- إرسال الحجز يتم عبر فتح تطبيق البريد على جهاز المستخدم (رابط `mailto`) مع تعبئة الطلب تلقائيا.
- الإرسال التلقائي بدون تدخل المستخدم يحتاج Backend وخدمة بريد، ويمكن إضافته لاحقا عند ربط قاعدة البيانات.

## حل مشكلة EMFILE في macOS

إذا ظهر الخطأ `EMFILE: too many open files, watch`:

1. استخدم Node LTS (20):

```bash
nvm install 20
nvm use 20
```

2. نظف ثم أعد التثبيت:

```bash
rm -rf node_modules package-lock.json
npm install
```

3. شغّل Metro مع كاش نظيف:

```bash
npx expo start -c
```

4. إذا استمر الخطأ، ارفع حد الملفات في نفس التيرمنال:

```bash
ulimit -n 65536
npx expo start -c
```

5. الأفضل على macOS: ثبّت `watchman` (مرة واحدة):

```bash
brew install watchman
```
